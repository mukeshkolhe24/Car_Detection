\## 📋 Overview



This project implements and compares two state-of-the-art object detection algorithms for detecting cars in images:

\- \*\*Faster R-CNN\*\* (Two-stage detector - High Accuracy)

\- \*\*YOLO\*\* (Single-stage detector - High Speed)



\## 📊 Results



| Metric | Faster R-CNN | YOLO |

|--------|--------------|------|

| mAP@0.5 | \*\*94.9%\*\* | 52.0% |

| Precision | \*\*95.0%\*\* | 52.0% |

| Recall | \*\*94.0%\*\* | 50.0% |

| F1-Score | \*\*0.945\*\* | 0.510 |

| Speed | 22 FPS | \*\*66 FPS\*\* |
